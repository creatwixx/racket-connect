# Racket Connect

A social app for finding padel buddies and organizing matches.

## Project Structure

```
racket-connect/
├── mobile/          # Flutter mobile app (iOS & Android)
├── web/             # React web app (coming soon)
└── README.md
```

## Mobile App (Flutter)

The mobile app is built with Flutter and supports iOS and Android.

### Setup

See `mobile/README.md` for mobile app setup instructions.

### Run Mobile App

```bash
cd mobile
flutter pub get
flutter run
```

## Web App (React)

The web app will be built with React (stack TBD).

### Setup

See `web/README.md` for web app setup instructions (coming soon).

## Features

- Create padel matches with location and time
- View all available matches
- Join/leave matches
- See match participants
- Dummy authentication (for demo)

### Mobile
